Horizontal Menu
Here is the typical HTML code for a standard vertical menu. As we will shortly see, this is exactly the same html that will be used in our horizontal menu.

<doctype html>
<html lang="en-US">
    <head>
        <meta charset="utf-8">
        <title>Horizontal Menu</title>
        <link rel="stylesheet" href="horz-menu.css" type="text/css" media="screen"/>
    </head>
    <body>
        <header>
            <nav>
                <ul>
                    <li><a href="item1.html">Item 1</a>
                        <ul>
                            <li><a href="item1-1.html">Submenu Item 1</a></li>
                            <li><a href="item1-2.html">Submenu Item 2</a></li>
                        </ul>
                    </li>
                    <li><a href="item2.html">Item 2</a></li>
                    <li><a href="item3.html">Item 3</a></li>
                    <li><a href="item4.html">Item 4</a></li>
                    <li><a href="item5.html">Item 5</a></li>
                </ul>
            </nav>
        </header>
        <main>
        </main>
    </body>
    <footer>
    </footer>
</html>


And here is the working menu the code produces. If you click on one of the menu items, the browser will attempt to take you to a non-existent page and display the "Page not found" error". You will need to press the "Back" button to return to this page. Again, what you see if a fairly standard vertical html menu.

To convert this to a horizontal menu, we don't make changes to the HTML, but instead use CSS to style the menu to the desired appearance. Here is the necessary CSS, annotated with explanations each line.

nav {
    font-size: 20px;
    //font-family: "TSCu-Comic", Georgia, sans-serif;
    padding-top: 0px;
}

nav a {
    color: red;
    padding: 0px 0px 0px 15px;
    text-decoration: none;
}

nav ul {
    padding: 0px 0px 0px 0px;
    list-style: none;
}

nav a:hover {
    color: yellow;
}

nav ul li
{
    position:relative;
    float:left;
    margin:0;
    padding:0
}

nav li {

    float: left;
    position: relative;
    padding-right: 100;
    display: block;
    //background-color: red;
}

nav li ul {
    display: none;
    position: absolute;
}

nav li ul li {
    clear: both;
    border-style: none;
}

nav li:hover ul {
    display: block;
    //background: #434343;
    height: auto;
}
