Browser Object Model
BOM Global Property Property Description Returned Value
window.location Current URL
window.screen Reference to Screen Object
window.screen.width Width of screen in pixels
window.screen.height Height of screen in pixels
window.screenX X-coordinate of cursor (in pixels) relative to upper-left corrner of screen
window.screenY Y-coordinate of cursor (in pixels) relative to upper-left corrner of screen
window.innerWidth Width of browser window in pixels
window.innerHeight Height of browser window in pixels
window.document Reference to current page object
window.history Reference to browser history object
window.history.length Number of pages in browser history



BOM Global Method Method Description
window.alert() Open an alert dialog displaying the method arguments
window.open() Open a new browser window using the method argument as the URL
window.print() Open the browser's print dialog for the current page
